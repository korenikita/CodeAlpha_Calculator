# CodeAlpha Calculator 🧮

A simple and responsive calculator web application built using **HTML, CSS, and JavaScript**.  
This project performs basic arithmetic operations with a clean and user-friendly interface.

---

## 🚀 Features

- Basic arithmetic operations: **Addition (+), Subtraction (−), Multiplication (×), Division (÷)**
- Real-time calculation display
- Clear (AC) and delete functionality
- Responsive and modern UI design
- Keyboard support (optional enhancement)
- Smooth user experience

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

## 💡 How to Use

1. Open `index.html` in any web browser
2. Use buttons or keyboard to perform calculations
3. Click **AC** to clear the screen
